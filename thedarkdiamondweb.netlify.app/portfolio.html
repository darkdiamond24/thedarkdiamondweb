Quote generator Kotlin script

I made a Kotlin script that generates random quotes. It uses class and object instances as well as Boolean statements. I used the when statement. There are 50 different possible quotes which are printed to the console.

Depending on the random number generated, the corresponding println statement for that number will be printed to the console. The script can be found on my Github. I will also have the script below so you can see and try out the script yourself.
